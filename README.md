# Blog (Jekyll)

Personal site built with the [Mediumish](https://github.com/wowthemesnet/mediumish-theme-jekyll) Jekyll theme.

## Run locally

```bash
bundle install
bundle exec jekyll serve --baseurl ""
```

## Comments (Giscus)

Comments use **[giscus](https://github.com/giscus/giscus)** (GitHub Discussions). Configure under `giscus:` in `_config.yml` — see [giscus.app](https://giscus.app). Use `comments: false` in front matter to disable on a page or post.
